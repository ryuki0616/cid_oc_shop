私たちのビジョン

CID OC Shopは、シンプルさと機能性を兼ね備えた商品を提供することで、 お客様の日常をより豊かにすることを目指しています。 モダンなデザインと高品質な製品で、新しいライフスタイルを提案します。

品質へのこだわり

全ての商品は厳選された素材を使用し、細部まで丁寧に作られています。 品質管理を徹底し、長く愛用できる製品だけをお届けします。 お客様の満足度が私たちの最優先事項です。

お客様サポート

購入前のご相談から、アフターサービスまで、専門スタッフが丁寧にサポートいたします。 ご不明な点がございましたら、お気軽にお問い合わせください。 お客様の声を大切にしています。

利用規約

CID OC Shopをご利用いただくにあたり、以下の利用規約をお読みください。

第1条（適用）

本利用規約（以下「本規約」といいます）は、CID OC Shop（以下「当ショップ」といいます）が 提供するサービス（以下「本サービス」といいます）の利用条件を定めるものです。 ユーザーの皆さま（以下「ユーザー」といいます）には、本規約に従って本サービスをご利用いただきます。

第2条（会員登録）

  1. 本サービスの利用を希望する方は、本規約に同意の上、当ショップの定める方法によって会員登録を申請するものとします。
  2. 当ショップは、登録申請者が以下のいずれかの事由に該当する場合、登録を拒否することがあります。
    • 登録情報に虚偽の記載がある場合
    • 過去に本規約に違反した者である場合
    • その他、当ショップが不適切と判断した場合

第3条（商品の購入）

ユーザーは、本サービスを通じて商品を購入する際、以下の事項に同意するものとします。

  1. 商品の価格は、当ショップのウェブサイトに表示された価格とします。
  2. 商品の在庫状況により、注文後に在庫切れとなる場合があります。
  3. 支払方法は、クレジットカード、銀行振込、コンビニ決済等、当ショップが指定する方法とします。
  4. 配送料は、購入金額や配送先により異なります。

第4条（配送）

  1. 商品の配送は、ご注文確定後、2〜3営業日以内に発送いたします。
  2. 配送先は日本国内に限ります。
  3. 天候や交通事情により、配送が遅延する場合があります。
  4. 5,000円以上のご注文で、全国送料無料となります。

第5条（返品・交換）

商品の返品・交換については、以下の条件に従うものとします。

  1. 商品到着後30日以内であれば、未開封・未使用の商品に限り返品可能です。
  2. 商品に欠陥がある場合、または配送中の破損等があった場合は、当ショップ負担で交換いたします。
  3. お客様都合による返品の場合、返送料はお客様負担となります。
  4. セール品や特別価格品は返品・交換の対象外となる場合があります。

第6条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為
  • 犯罪行為に関連する行為
  • 当ショップのサーバーまたはネットワークの機能を破壊したり、妨害したりする行為
  • 当ショップのサービスの運営を妨害するおそれのある行為
  • 他のユーザーに関する個人情報等を収集または蓄積する行為
  • 他のユーザーに成りすます行為
  • 当ショップが許諾しない本サービス上での宣伝、広告、勧誘、または営業行為
  • その他、当ショップが不適切と判断する行為

第7条（個人情報の取扱い）

当ショップは、本サービスの利用によって取得する個人情報については、 当ショップのプライバシーポリシーに従い適切に取り扱うものとします。 個人情報は、商品の配送、お問い合わせへの対応、サービス向上のためにのみ使用し、 第三者に提供することはありません。

第8条（サービスの変更・停止）

  1. 当ショップは、ユーザーに事前に通知することなく、本サービスの内容を変更または停止することができるものとします。
  2. 当ショップは、本サービスの変更または停止により、ユーザーまたは第三者が被ったいかなる不利益または損害についても、一切の責任を負わないものとします。

第9条（免責事項）

  1. 当ショップは、本サービスに関して、ユーザーと他のユーザーまたは第三者との間において生じた取引、連絡または紛争等について一切責任を負いません。
